# Features

Esta pasta organiza o codigo pela leitura do produto.

`app/` continua sendo a camada de rotas do Next.js. As features guardam catalogos, regras, hooks, componentes e servicos que tornam cada area do sistema facil de encontrar.

Regra: uma tela nao deve ser duplicada por perfil. A feature deve expor a mesma estrutura e deixar visibilidade, acesso e acoes serem filtrados por perfil/permissao.
