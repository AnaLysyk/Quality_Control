# Fluxo profissional da Agenda

- Meus agendamentos deve funcionar como calendário pessoal.
- Agendamento geral deve ter seletor de empresa para Líder TC, Suporte Técnico e Usuário TC.
- Usuário de empresa visualiza apenas a própria empresa.
- Clique no dia expande a lista de reuniões do dia.
- Clique na reunião abre detalhes, participantes, empresa, projeto, responsável, duração e status.
- Status esperados: vai acontecer, realizada, não realizada, em risco, bloqueada.
- Reunião criada pelo Chat/Brain deve gerar registro automático de agenda, com participantes e horário.
- Agenda deve notificar 5 minutos antes.
- Agendamentos gerais devem ser filtráveis por empresa, projeto e usuário.
