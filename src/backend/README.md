# Backend

Esta pasta organiza a leitura server-side por dominio.

As rotas continuam em `app/api/`. A migracao para ca deve acontecer por etapas, trazendo primeiro regras puras, servicos e adaptadores pequenos. Nenhuma API deve ser movida sem mapear importacoes, permissao e testes.

Dominios previstos:

- admin
- empresas
- suporte
- brain
- automacoes
- repositorio-testes
- integracoes
